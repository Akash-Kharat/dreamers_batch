Imortant Notes
###############

Q. Does CSS transform functions affect box model properties?

A. No, it doesn't. Therefore we can safely use them without affecting the spacing of other neighbouring elements.


Q. How much transition-duration is ususally suggested for UI effects.?
A. Usually 200 - 300ms works best.

Q. When should we use % format in @keyframes instead of from-to format for animations?
A. Whenever we want multiple steps in the animation.
