Important Notes
################################

Q. Which image related CSS can we use to preserve aspect ratio of the image?

A: We can use both 'object-fit:cover' and 'object-fit:contain'

    cover - will do cropping but will fill up completely
    contain - will not do cropping but image will be shrinked.



Q. How to position an item in the center both vertically and horizontally using flexbox(display:flex) ?

A. Use this CSS onto the Flexbox Container
    display: flex;
    justify-content: center;
    align-items: center;
